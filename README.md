# MUJConnect – Home Services Booking Platform

A modern **home services booking platform** inspired by Urban Company, built using **React, Vite, TypeScript, and Tailwind CSS**.  
MUJConnect allows users to browse services, book appointments, and manage bookings seamlessly with a clean and responsive UI.

---

## Features

-  **Home Page**
   Hero section with call-to-action  
   Category grid  
   Featured services  

-  **Services Listing**
   Tab-based category filtering  
   Responsive service cards  

-  **Service Details**
   Detailed service view  
   Add-ons selection  

-  **Shopping Cart**
   Side drawer cart  
   Add/remove items dynamically  

-  **Checkout Flow**
   Address input  
   Date & time selection  
   Order summary  

-  **Responsive Design**
   Mobile-first approach  
   Optimized for all devices
