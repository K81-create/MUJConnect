# 🌐 MUJConnect — Urban Home Services Platform

A modern, production-ready web application inspired by Urban Company that allows users to discover, book, and manage home services seamlessly.

Built with a clean UI, responsive design, and scalable architecture, MUJConnect delivers a smooth booking experience from service discovery to checkout.

---

## 🚀 Features

### 🏠 Home Page
- Hero section with headline  
- Category grid (6 categories)  
- Featured services section  
- Fully responsive layout  

### 🔍 Services Listing
- Category-based filtering (Women, Men, Repair, Cleaning)  
- Search functionality  
- Service cards with ratings, duration, and pricing  
- URL-based filtering  

### 📄 Service Details
- Detailed service information  
- Add-ons selection  
- Dynamic price calculation  
- Sticky pricing sidebar  
- “Add to Cart” & “Book Now”  

### 🛒 Shopping Cart
- Side drawer cart (Sheet UI)  
- Add/remove services  
- Add-ons display  
- Total price calculation  

### 💳 Checkout Flow
- Address input form  
- Date picker & time slot selection  
- Order summary  
- Form validation  
- Booking confirmation dialog  

### 🎨 UI & Navigation
- Sticky navbar with search & location  
- Cart badge indicator  
- Clean, modern design  
- Mobile-first responsive layout
